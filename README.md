# Hotel Room Booking Management System

This project is a DBMS-based hotel room booking system that allows hotel managers to manage rooms and bookings efficiently.

## Technologies Used
- HTML
- CSS
- JavaScript
- Python (Flask)
- MySQL

## Features
- Add hotel rooms with different room types
- Online and offline booking support
- Aadhar verification for online booking
- Automatic room availability checking
- Booking records display

## Database Tables
- room_type
- room
- customer
- booking

## How to Run the Project
1. Start MySQL server
2. Run the Flask server:
   ```bash
   python server.py
